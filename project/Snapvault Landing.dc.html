Como funciona

Três passos.
Zero preocupação.

1

Adicione seu banco

Informe host, usuário e senha no painel. Testamos a conexão antes de salvar qualquer dado.

2

Instale o agente

Um comando npm no servidor do cliente. O agente configura o backup diário e envia confirmação.

3

Relaxe

Backup todo dia às 2h. Se falhar, alerta imediato. Se precisar restaurar, download em 1 clique.

Instalação

Um comando. Pronto.

bash — servidor-do-cliente
# instala o agente no servidor
$ npm install -g @snapvault/agent
# conecta com o token da conta
$ snapvault init --token sv_live_xk9mPq3...
